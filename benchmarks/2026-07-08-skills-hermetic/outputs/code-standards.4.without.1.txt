```python
def apply_updates(prices, feed_lines):
    for line in feed_lines:
        try:
            sku, price = line.split(",")
            prices[sku] = float(price)
        except ValueError:
            continue
    return prices
```

Change: each line is now processed inside a `try/except ValueError`, which catches both a bad split (wrong number of fields) and a bad `float()` conversion. Malformed lines are skipped instead of crashing the batch; everything else is untouched.
