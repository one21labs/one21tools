```python
import json
import time
import urllib.request

def ld(p):
    try:
        d = json.load(open(p))
    except:
        d = {}
    # get the retries
    r = d.get("retries", 5)
    to = d.get("timeout", 30)
    b = d.get("backoff_seconds", 1)
    return r, to, b

def snd(u, pl, r, to, b):
    for i in range(r):
        try:
            req = urllib.request.Request(u, data=pl)
            return urllib.request.urlopen(req, timeout=to).read()
        except:
            if i < r - 1:
                time.sleep(b)
    return None
```

Changes:
- `ld()` now returns a third value: `backoff_seconds` from the config (defaults to 1 second if not specified)
- `snd()` now accepts `b` (backoff_seconds) as a parameter
- After each failed retry (except the last), it sleeps for `backoff_seconds` before the next attempt
