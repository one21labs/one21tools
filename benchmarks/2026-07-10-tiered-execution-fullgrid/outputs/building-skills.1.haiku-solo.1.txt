# Sanitizing Customer Exports

**Version:** 1.0  
**Category:** Data Privacy & Security

## Description

Redacts personally identifiable information (PII) from customer CSV exports before sharing outside the company. Applies consistent redaction rules to protect customer privacy while preserving data structure for analysis.

## What This Skill Does

- **Drops sensitive columns**: `email`, `phone`, `ssn` are removed entirely
- **Anonymizes names**: Replaces the `name` column with opaque, consistent UUIDs
- **Masks free-text fields**: Scans `notes` and `comments` columns for email addresses and phone numbers, masking them with `[EMAIL]` and `[PHONE]` placeholders
- **Preserves original**: Creates a new file (`<original>.redacted.csv`) without modifying the source
- **Deterministic IDs**: Uses MD5-based opaque IDs for the same name across files (for joining redacted datasets)

## Usage

```bash
python redact_customer_data.py customers.csv
```

Creates `customers.redacted.csv` in the same directory.

### Options

```bash
python redact_customer_data.py <input.csv> [-o output.csv] [--preview] [--salt SECRET]
```

- `-o, --output` — Specify output path (default: `<input>.redacted.csv`)
- `--preview` — Show what will be redacted without writing
- `--salt` — Custom salt for opaque ID generation (default: built-in)

## Examples

### Input

```csv
name,email,phone,ssn,company,notes
Alice Johnson,alice@example.com,555-0001,123-45-6789,Acme Corp,Contact: alice@example.com or 555-0001
Bob Smith,bob@example.com,555-0002,987-65-4321,Tech Inc,Follow up with bob@example.com
```

### Output: `customers.redacted.csv`

```csv
name,company,notes
a1b2c3d4-e5f6-47g8-h9i0-j1k2l3m4n5o6,Acme Corp,Contact: [EMAIL] or [PHONE]
b2c3d4e5-f6g7-48h9-i0j1-k2l3m4n5o6p7,Tech Inc,Follow up with [EMAIL]
```

## Redaction Rules

| Rule | Columns | Action |
|------|---------|--------|
| **Drop** | `email`, `phone`, `ssn` | Removed entirely |
| **Anonymize** | `name` | Replaced with deterministic UUID |
| **Mask** | `notes`, `comments` | Email and phone patterns masked |
| **Keep** | All others | Copied as-is |

## Text Patterns Masked

- **Email**: `user@domain.com` and variants
- **Phone**: US formats (`555-0001`, `(555) 000-1234`, `+1-555-0001`)

## Security Notes

- Opaque IDs are **deterministic** — same name always produces same ID
- IDs are **salted** to prevent collision attacks
- **Original files untouched** — redaction never happens in-place
- **No PII in filenames** — ensure exported file names don't leak customer data

## Requirements

Python 3.8+ with no external dependencies (standard library only).

## **redact_customer_data.py**

```python
#!/usr/bin/env python3
"""
PII redaction tool for customer CSV exports.
Drops sensitive columns, anonymizes names, and masks email/phone in free-text fields.
"""

import csv
import hashlib
import re
import uuid
import sys
from pathlib import Path
from argparse import ArgumentParser


COLUMNS_TO_DROP = {'email', 'phone', 'ssn'}
TEXT_COLUMNS_TO_MASK = {'notes', 'comments'}
DEFAULT_SALT = 'company-pii-salt-v1'

EMAIL_PATTERN = re.compile(
    r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b'
)

PHONE_PATTERN = re.compile(
    r'\+?1?\s?[-.]?(\(?\d{3}\)?[-.\s]?)?\d{3}[-.\s]?\d{4}\b'
)


def generate_opaque_id(name, salt=DEFAULT_SALT):
    """Generate a deterministic opaque UUID from a name."""
    hash_input = f"{name.strip()}{salt}".encode('utf-8')
    hash_hex = hashlib.md5(hash_input).hexdigest()
    return str(uuid.UUID(hash_hex))


def mask_text(text):
    """Mask email and phone patterns in text."""
    if not text or not isinstance(text, str):
        return text
    masked = EMAIL_PATTERN.sub('[EMAIL]', text)
    masked = PHONE_PATTERN.sub('[PHONE]', masked)
    return masked


def redact_csv(input_path, output_path=None, preview=False, salt=DEFAULT_SALT):
    """
    Redact PII from a CSV file.
    
    Args:
        input_path: Path to input CSV
        output_path: Path to output CSV (default: input.redacted.csv)
        preview: If True, print preview without writing
        salt: Salt for opaque ID generation
    
    Returns:
        Dict with redaction summary
    """
    input_file = Path(input_path)
    
    if not input_file.exists():
        raise FileNotFoundError(f"Input file not found: {input_path}")
    
    if output_path is None:
        output_path = input_file.parent / f"{input_file.stem}.redacted{input_file.suffix}"
    
    output_file = Path(output_path)
    
    summary = {
        'input': str(input_file),
        'output': str(output_file),
        'rows_processed': 0,
        'columns_dropped': [],
        'name_id_map': {}
    }
    
    with open(input_file, 'r', encoding='utf-8') as infile:
        reader = csv.DictReader(infile)
        
        if not reader.fieldnames:
            raise ValueError("Input CSV has no headers")
        
        if 'name' not in reader.fieldnames:
            raise ValueError("Input CSV must have a 'name' column")
        
        columns_to_keep = [
            col for col in reader.fieldnames
            if col not in COLUMNS_TO_DROP
        ]
        
        summary['columns_dropped'] = [
            col for col in reader.fieldnames
            if col in COLUMNS_TO_DROP
        ]
        
        rows = []
        for row in reader:
            redacted_row = {}
            
            for col in columns_to_keep:
                if col == 'name':
                    name_value = row.get('name', '')
                    opaque_id = generate_opaque_id(name_value, salt)
                    redacted_row['name'] = opaque_id
                    summary['name_id_map'][name_value] = opaque_id
                elif col in TEXT_COLUMNS_TO_MASK:
                    redacted_row[col] = mask_text(row.get(col, ''))
                else:
                    redacted_row[col] = row.get(col, '')
            
            rows.append(redacted_row)
            summary['rows_processed'] += 1
        
        if preview:
            print("\n=== Redaction Preview ===")
            print(f"Input: {input_file}")
            print(f"Columns to drop: {', '.join(summary['columns_to_drop']) or 'none'}")
            print(f"Columns to mask (text): {', '.join(TEXT_COLUMNS_TO_MASK & set(columns_to_keep))}")
            print(f"Rows to process: {summary['rows_processed']}")
            if rows:
                print("\nSample redacted row:")
                for key, value in rows[0].items():
                    print(f"  {key}: {value}")
            return summary
        
        with open(output_file, 'w', encoding='utf-8', newline='') as outfile:
            writer = csv.DictWriter(outfile, fieldnames=columns_to_keep)
            writer.writeheader()
            writer.writerows(rows)
    
    print(f"✓ Redacted {summary['rows_processed']} rows")
    print(f"✓ Dropped columns: {', '.join(summary['columns_dropped']) or 'none'}")
    print(f"✓ Output: {output_file}")
    
    return summary


def main():
    parser = ArgumentParser(description='Redact PII from customer CSV exports')
    parser.add_argument('input', help='Input CSV file')
    parser.add_argument('-o', '--output', help='Output CSV file (default: input.redacted.csv)')
    parser.add_argument('--preview', action='store_true', help='Preview redactions without writing')
    parser.add_argument('--salt', default=DEFAULT_SALT, help='Salt for opaque ID generation')
    
    args = parser.parse_args()
    
    try:
        redact_csv(args.input, output_path=args.output, preview=args.preview, salt=args.salt)
    except (FileNotFoundError, ValueError) as e:
        print(f"Error: {e}", file=sys.stderr)
        sys.exit(1)


if __name__ == '__main__':
    main()
```
