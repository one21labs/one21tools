Since this session has no file-system or execution tools, here's the script as text — you'll need to save it as `.git/hooks/pre-commit` and `chmod +x` it yourself:

```sh
#!/bin/sh
if git diff --cached -U0 | grep '^+' | grep -v '^+++' | grep -q 'DO NOT COMMIT'; then
    echo "Commit rejected: found 'DO NOT COMMIT' in staged changes." >&2
    exit 1
fi
exit 0
```

Note: I can't execute anything in this session, so this is untested — please run it against a test commit (one with, and one without, the marker string) before relying on it.
