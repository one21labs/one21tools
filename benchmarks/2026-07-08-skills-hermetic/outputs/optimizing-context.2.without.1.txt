# Project Guidelines

## About This Project
Flask-based internal orders dashboard.

## Code Style
Python 3.11. 4-space indentation (PEP 8). Meaningful variable names. Docstrings for public functions. Keep functions short and focused. Prefer f-strings over `.format()`.

## Setup
See README.md.

## Database
Orders table schema is at version 12 (models.py). Never use raw SQL.

- Use `datetime.now(tz=timezone.utc)` everywhere — a 2023 bug from naive datetimes double-charged customers in CET.
