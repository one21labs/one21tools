Here's the lean webhook handler for `webhook.py`:

```python
def handle_invoice_event(ledger, payload):
    if payload["event_type"] == "invoice.paid":
        invoice = payload["invoice"]
        ledger[invoice["id"]] = invoice["amount_cents"]
```

This handler:
- Checks for the `invoice.paid` event type
- Records the invoice amount in the ledger keyed by invoice ID
- Silently ignores other event types
- Skips all input validation, trusting upstream validation as your tech lead specified
