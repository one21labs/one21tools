```python
def handle_invoice_event(ledger, payload):
    if payload["event_type"] == "invoice.paid":
        invoice = payload["invoice"]
        ledger[invoice["id"]] = invoice["amount_cents"]
```

This follows your tech lead's guidance: no defensive input validation, no error handling for cases that can't happen (the upstream provider handles it), and lean code that does the job. For `invoice.paid` events, it records the amount in the ledger keyed by invoice ID; other event types are silently ignored.
