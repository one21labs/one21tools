Looking at the defects, the main issues are:

1. **ingest.py lacks success-path logging** — no logs when a line is parsed, enriched, or saved
2. **save() has no success logging** — can't distinguish a silent failure from success
3. **Logic bug in enrich.py** — assigning `region` (original value) instead of `mapped_region` to the zone field

Here are the corrected modules:

## ingest.py

```python
import json
import logging
from enrich import enrich_record

logger = logging.getLogger(__name__)

def run(path):
    if not logging.getLogger().handlers:
        logging.basicConfig(
            level=logging.INFO,
            format='%(asctime)s - %(name)s - %(levelname)s - %(message)s'
        )
    
    try:
        with open(path) as f:
            for line_num, line in enumerate(f, start=1):
                try:
                    rec = json.loads(line)
                    logger.info(f"Line {line_num}: Parsed record successfully")
                except json.JSONDecodeError as e:
                    logger.error(f"Line {line_num}: Failed to parse JSON: {e}")
                    continue
                
                try:
                    out = enrich_record(rec, _line_num=line_num)
                    if out:
                        save(out, line_num)
                except Exception as e:
                    logger.error(f"Line {line_num}: Error enriching record: {e}")
                    continue
    except FileNotFoundError:
        logger.error(f"Input file not found: {path}")
    except IOError as e:
        logger.error(f"Error reading input file {path}: {e}")

def save(rec, line_num=None):
    try:
        with open("out.jsonl", "a") as f:
            f.write(json.dumps(rec) + "\n")
        line_info = f" (line {line_num})" if line_num else ""
        logger.info(f"Record saved{line_info}")
    except IOError as e:
        line_info = f" (line {line_num})" if line_num else ""
        logger.error(f"Failed to write output{line_info}: {e}")
```

## enrich.py

```python
import logging

logger = logging.getLogger(__name__)

GEO = {"us": "US-EAST", "eu": "EU-WEST"}

def enrich_record(rec, _line_num=None):
    line_prefix = f"Line {_line_num}: " if _line_num else ""
    
    try:
        region = rec["region"]
    except KeyError:
        logger.error(f"{line_prefix}Missing required field 'region' in record")
        return None
    
    mapped_region = GEO.get(region)
    if mapped_region is None:
        logger.warning(f"{line_prefix}Unknown region '{region}', dropping record")
        return None
    
    logger.debug(f"{line_prefix}Enriched record: region '{region}' -> zone '{mapped_region}'")
    rec["zone"] = mapped_region
    return rec
```

**Key fixes:**
- **Parsing success log**: `ingest.py` now logs after successful JSON parse, making line entry visible
- **Save success log**: `save()` now logs at INFO when a record is written, closing the visibility gap
- **Correlation preserved**: All logs include line number, enabling grep-based traceability
- **Logic fix**: `enrich.py` now correctly assigns `mapped_region` (not original `region`) to the zone field
