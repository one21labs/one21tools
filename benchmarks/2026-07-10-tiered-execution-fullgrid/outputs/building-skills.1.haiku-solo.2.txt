# Sanitizing Customer Exports

**Version:** 1.0  
**Category:** Data Privacy & Compliance  

## Overview

This skill safely redacts personally identifiable information (PII) from customer CSV exports before sharing them outside the company. It automatically applies consistent redaction rules and outputs a sanitized copy without modifying the original file.

## Features

- **Automatic PII removal**: Drops sensitive columns (email, phone, SSN) entirely
- **Name masking**: Replaces customer names with opaque IDs for reference
- **Text scanning**: Detects and masks email addresses and phone numbers in free-text fields
- **Safe processing**: Original file is never modified; redacted copy is always written separately

### Columns Dropped
- `email`
- `phone`
- `ssn`

### Columns Transformed
- `name` → replaced with `customer_id` (opaque reference)

### Columns Scanned & Masked
- `notes`
- `comments`
- Any other free-text columns get email and phone patterns masked

## **sanitize_exports.py**

```python
#!/usr/bin/env python3
"""
Sanitize customer CSV exports by removing and masking PII.
"""
import csv
import re
import sys
from pathlib import Path


COLUMNS_TO_DROP = {'email', 'phone', 'ssn'}
TEXT_COLUMNS_TO_SCAN = {'notes', 'comments'}

EMAIL_PATTERN = re.compile(r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b')
PHONE_PATTERN = re.compile(r'\b(?:\d{3}[-.]?\d{3}[-.]?\d{4}|\(\d{3}\)\s?\d{3}[-.]?\d{4}|\d{10})\b')


def mask_pii_in_text(text):
    """Mask email addresses and phone numbers in free text."""
    if not isinstance(text, str):
        return text
    
    text = EMAIL_PATTERN.sub('[REDACTED_EMAIL]', text)
    text = PHONE_PATTERN.sub('[REDACTED_PHONE]', text)
    return text


def sanitize_csv(input_file):
    """
    Read a CSV file, redact PII, and write a sanitized copy.
    
    Args:
        input_file (str): Path to the input CSV file
        
    Returns:
        str: Path to the output redacted CSV file
    """
    input_path = Path(input_file)
    if not input_path.exists():
        raise FileNotFoundError(f"Input file not found: {input_file}")
    
    output_path = input_path.parent / f"{input_path.stem}.redacted{input_path.suffix}"
    
    with open(input_path, 'r', encoding='utf-8') as infile:
        reader = csv.DictReader(infile)
        
        if reader.fieldnames is None:
            raise ValueError("CSV file is empty or invalid")
        
        fieldnames_lower = {name: name for name in reader.fieldnames}
        columns_to_keep = [
            name for name in reader.fieldnames
            if name.lower() not in COLUMNS_TO_DROP
        ]
        
        redacted_rows = []
        for row_num, row in enumerate(reader, start=1):
            redacted_row = {}
            
            for col in columns_to_keep:
                if col.lower() == 'name':
                    redacted_row['customer_id'] = f'customer_id_{row_num}'
                elif col.lower() in TEXT_COLUMNS_TO_SCAN:
                    redacted_row[col] = mask_pii_in_text(row.get(col, ''))
                else:
                    redacted_row[col] = row.get(col, '')
            
            redacted_rows.append(redacted_row)
    
    output_fieldnames = [col if col.lower() != 'name' else 'customer_id' 
                         for col in columns_to_keep]
    
    with open(output_path, 'w', encoding='utf-8', newline='') as outfile:
        writer = csv.DictWriter(outfile, fieldnames=output_fieldnames)
        writer.writeheader()
        writer.writerows(redacted_rows)
    
    return str(output_path)


if __name__ == '__main__':
    if len(sys.argv) < 2:
        print("Usage: python sanitize_exports.py <input_csv_file>")
        sys.exit(1)
    
    input_file = sys.argv[1]
    output_file = sanitize_csv(input_file)
    print(f"Redacted export written to: {output_file}")
```
