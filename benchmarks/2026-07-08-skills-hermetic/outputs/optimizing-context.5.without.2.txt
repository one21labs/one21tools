## Style Rules
- Prefer `const` over `let`.

## API Auth — IMPORTANT, do not use Bearer auth
Requests to `/api/*` must carry an `X-Client-Sig` header: an HMAC-SHA256 signature
of the request body, generated using the key from `scripts/sign.mjs`. This is a
custom signing scheme, NOT standard Bearer token auth. Do not write
`Authorization: Bearer ...` code for these routes — it will fail. Full details in
`docs/auth.md`.

## Tests
- `pnpm test` — launches real Chromium instances and requires the dev server
  running. Use for full/CI-style runs.
- `pnpm test:unit` — fast, no browser/dev server needed. Use this for iteration.

## Onboarding
1. Clone the repo.
2. `pnpm install`
3. Copy `.env.example` to `.env`.
4. `pnpm dev`
5. Open `localhost:3000`.
