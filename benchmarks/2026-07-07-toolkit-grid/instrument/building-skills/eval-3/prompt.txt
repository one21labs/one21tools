# Reviewing Terraform Plans

Before approving any infrastructure change, run `terraform plan -out=tfplan` and review the full plan output.

Flag as blocking:
- Any destruction or replacement of stateful resources (databases, volumes, buckets)
- Security group or IAM changes that widen access
- Changes to resources outside the directory named in the ticket

Post the review as a comment listing each risky change with its resource address and why it is risky.
