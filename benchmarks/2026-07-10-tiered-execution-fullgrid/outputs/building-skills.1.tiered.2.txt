# Sanitizing Customer Exports

Before any customer CSV export leaves the company, PII must be redacted. This skill applies deterministic, company-approved rules to produce a safe-to-share copy.

## What This Skill Does

Given a customer CSV export, produces a redacted copy (`<original>.redacted.csv`) in the same directory using these fixed rules:

1. **Dropped columns** (removed entirely, not blanked):
   - `email`
   - `phone`
   - `ssn`
   - (Column names matched case-insensitively; column is skipped gracefully if absent)

2. **Pseudonymized**: `name` column replaced with an opaque customer ID (`CUST_00001`, `CUST_00002`, etc.) in order of appearance.

3. **Masked free text**: `notes` and `comments` columns scanned for embedded email addresses and phone number substrings; matching patterns replaced with `[REDACTED_EMAIL]` or `[REDACTED_PHONE]`.

4. **Preserved content**: All other columns and free-text content outside the masked patterns pass through unchanged.

5. **Safety**: The original file is never modified; output is always written with the `.redacted.csv` suffix.

## Inputs & Outputs

**Input:** A CSV file with a header row (any encoding ISO-8859-1 or UTF-8) containing customer data.

**Output:** A new file `<original>.redacted.csv` in the same directory, with PII removed per the rules above.

**Assumptions:**
- CSV has a header row (required).
- Standard CSV format: commas separate fields; double quotes escape commas within field values.
- UTF-8 or ISO-8859-1 encoding (detected automatically; ISO-8859-1 is fallback).

## How to Use

1. **Locate your customer export** (e.g. `customers.csv`).
2. **Invoke the skill**: Pass the file path to the `sanitize.py` script.
   ```bash
   python sanitize.py customers.csv
   ```
3. **Review the output**: A new file `customers.redacted.csv` appears in the same directory.
4. **Audit**: Open the `.redacted.csv` file and spot-check that PII is masked correctly (names → customer IDs, emails/phones masked in free text, sensitive columns gone).
5. **Share safely**: The `.redacted.csv` file is now safe to share with external parties.

#!/usr/bin/env python3
"""
Redact PII from customer CSV exports.
Usage: python sanitize.py <input.csv>
Output: <input>.redacted.csv (written to same directory as input)
"""

import csv
import re
import sys
from pathlib import Path


EMAIL_PATTERN = re.compile(r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b')
PHONE_PATTERN = re.compile(r'(?:\+?1[-.\s]?)?\(?[0-9]{3}\)?[-.\s]?[0-9]{3}[-.\s]?[0-9]{4}\b')

COLUMNS_TO_DROP = {'email', 'phone', 'ssn'}
COLUMNS_TO_MASK_TEXT = {'notes', 'comments'}


def sanitize_text(text):
    """Mask emails and phone numbers in free-text fields."""
    text = EMAIL_PATTERN.sub('[REDACTED_EMAIL]', text)
    text = PHONE_PATTERN.sub('[REDACTED_PHONE]', text)
    return text


def redact_csv(input_path):
    """Read CSV, apply redaction rules, write .redacted.csv output."""
    input_path = Path(input_path)
    if not input_path.exists():
        print(f"Error: File not found: {input_path}", file=sys.stderr)
        sys.exit(1)

    output_path = input_path.with_stem(input_path.stem + '.redacted')

    try:
        with open(input_path, 'r', encoding='utf-8', newline='') as infile:
            reader = csv.DictReader(infile)
            if not reader.fieldnames:
                print("Error: CSV has no header row", file=sys.stderr)
                sys.exit(1)

            fieldnames_lower = [f.lower() for f in reader.fieldnames]
            
            # Determine which columns to keep and which to mask
            keep_indices = []
            mask_text_indices = []
            for i, fname in enumerate(fieldnames_lower):
                if fname not in COLUMNS_TO_DROP:
                    keep_indices.append(i)
                    if fname in COLUMNS_TO_MASK_TEXT:
                        mask_text_indices.append(i)

            output_fieldnames = [reader.fieldnames[i] for i in keep_indices]
            
            # Track customer IDs for name pseudonymization
            name_index = None
            for i, fname in enumerate(fieldnames_lower):
                if fname == 'name':
                    name_index = i
                    break
            
            cust_id_counter = 1
            rows = []
            
            for row in reader:
                output_row = {}
                for i, orig_idx in enumerate(keep_indices):
                    field_name = reader.fieldnames[orig_idx]
                    value = row[field_name]
                    
                    # Pseudonymize name column
                    if orig_idx == name_index:
                        value = f"CUST_{cust_id_counter:05d}"
                        cust_id_counter += 1
                    # Mask PII in text columns
                    elif orig_idx in mask_text_indices:
                        value = sanitize_text(value)
                    
                    output_row[field_name] = value
                rows.append(output_row)

        # Write output file
        with open(output_path, 'w', encoding='utf-8', newline='') as outfile:
            writer = csv.DictWriter(outfile, fieldnames=output_fieldnames)
            writer.writeheader()
            writer.writerows(rows)

        print(f"Redacted CSV written to: {output_path}")

    except UnicodeDecodeError:
        # Fallback to ISO-8859-1 if UTF-8 fails
        with open(input_path, 'r', encoding='iso-8859-1', newline='') as infile:
            reader = csv.DictReader(infile)
            if not reader.fieldnames:
                print("Error: CSV has no header row", file=sys.stderr)
                sys.exit(1)

            fieldnames_lower = [f.lower() for f in reader.fieldnames]
            keep_indices = []
            mask_text_indices = []
            for i, fname in enumerate(fieldnames_lower):
                if fname not in COLUMNS_TO_DROP:
                    keep_indices.append(i)
                    if fname in COLUMNS_TO_MASK_TEXT:
                        mask_text_indices.append(i)

            output_fieldnames = [reader.fieldnames[i] for i in keep_indices]
            
            name_index = None
            for i, fname in enumerate(fieldnames_lower):
                if fname == 'name':
                    name_index = i
                    break
            
            cust_id_counter = 1
            rows = []
            
            for row in reader:
                output_row = {}
                for i, orig_idx in enumerate(keep_indices):
                    field_name = reader.fieldnames[orig_idx]
                    value = row[field_name]
                    
                    if orig_idx == name_index:
                        value = f"CUST_{cust_id_counter:05d}"
                        cust_id_counter += 1
                    elif orig_idx in mask_text_indices:
                        value = sanitize_text(value)
                    
                    output_row[field_name] = value
                rows.append(output_row)

        with open(output_path, 'w', encoding='utf-8', newline='') as outfile:
            writer = csv.DictWriter(outfile, fieldnames=output_fieldnames)
            writer.writeheader()
            writer.writerows(rows)

        print(f"Redacted CSV written to: {output_path}")


if __name__ == '__main__':
    if len(sys.argv) < 2:
        print(f"Usage: {sys.argv[0]} <input.csv>", file=sys.stderr)
        sys.exit(1)
    redact_csv(sys.argv[1])
```
