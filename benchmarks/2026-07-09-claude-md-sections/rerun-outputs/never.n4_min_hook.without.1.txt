```bash
#!/bin/sh
if git diff --cached | grep -q 'DO NOT COMMIT'; then
    echo "Commit rejected: found 'DO NOT COMMIT' in staged changes."
    exit 1
fi
```

Save as `.git/hooks/pre-commit` and `chmod +x` it. Note: since this is a headless session I can't create or chmod the file myself — you'll need to do that locally.
