```markdown
# 📊 CSV Cleanup

## 🔧 Cleanup Steps

1. Load with pandas: dtype=str, keep_default_na=False.
2. Report row count, duplicate count, and columns with mixed formats before changing anything.
3. Ask before dropping rows; never silently delete data.
4. Write the cleaned file next to the original with a .cleaned.csv suffix.
```
