## API Authentication

**Critical:** Requests to `/api/*` must carry an `X-Client-Sig` header: an HMAC-SHA256 of the request body using the key from `scripts/sign.mjs`. This is custom, not Bearer auth. Claude frequently writes standard Bearer code here and it always fails. See `docs/auth.md` for details.

- `pnpm test` launches real Chromium instances and requires the dev server running
