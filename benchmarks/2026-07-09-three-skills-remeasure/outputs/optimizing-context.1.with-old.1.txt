# checkout-service

Next.js 14 (App Router) + TypeScript + Tailwind + Prisma/Postgres. Owned by the payments squad
(rewritten 2023 by the platform team from a 2021 Rails prototype).

## Key Decisions

- All monetary amounts are integer cents. **Never use floats for money** — a 2024
  float-rounding incident corrupted invoices.
- Style deviations from typical TypeScript (full guide: docs/style-guide.md):
  - No default exports.
  - Zod schemas are colocated next to their route handlers.

## Common Mistakes

- `npm test` runs the Playwright E2E suite and requires a running dev server — it will fail
  otherwise. For unit tests use `npm run test:unit`.
- Never run `prisma migrate` directly — it breaks the shadow database CI relies on. Always
  migrate via `./scripts/migrate.sh`.
- API routes authenticate with a custom `X-Session-Token` header, not standard Bearer tokens.
  Read docs/api.md before touching auth code.

## Deploys

Same procedure as our other repos: merge to main → wait for CI green → `npm run build:prod` →
`./deploy staging` → smoke-test `/healthz` → page on-call if p95 latency > 500ms →
`./deploy prod` → announce in #releases.

## Where to Look

- docs/api.md — auth scheme details
- docs/style-guide.md — full style guide
- README.md — local onboarding/setup
