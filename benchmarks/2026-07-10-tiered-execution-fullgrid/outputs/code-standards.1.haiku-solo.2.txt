```python
import json
import time
import urllib.request

def ld(p):
    try:
        d = json.load(open(p))
    except:
        d = {}
    r = d.get("retries", 5)
    to = d.get("timeout", 30)
    bs = d.get("backoff_seconds", 1)
    return r, to, bs

def snd(u, pl, r, to, bs):
    for i in range(r):
        try:
            req = urllib.request.Request(u, data=pl)
            return urllib.request.urlopen(req, timeout=to).read()
        except:
            if i < r - 1:
                time.sleep(bs)
    return None
```

Changes made:
- `ld()` now reads and returns `backoff_seconds` from config with a default of 1 second
- `snd()` accepts `bs` parameter and sleeps between retry attempts (but not after the final attempt)
