# Sanitizing Customer Exports

Automatically redact PII (Personally Identifiable Information) from customer CSV files before sharing them outside the company.

## What it does

This skill processes customer CSV exports and creates a redacted copy with sensitive information masked:

- **Drops sensitive columns entirely**: email, phone, ssn
- **Replaces names with opaque IDs**: The `name` column is replaced with a hash-based identifier
- **Masks PII in free-text fields**: Scans `notes` and `comments` columns for email addresses and phone numbers, replacing them with `[REDACTED]`
- **Preserves original files**: Never modifies the original CSV; writes output as `<original>.redacted.csv`

## How to use

```bash
python redact.py path/to/customers.csv
```

This generates `customers.redacted.csv` in the same directory.

## Input format

A CSV file with standard customer data. These columns will be processed:

- **Dropped**: `email`, `phone`, `ssn`
- **Transformed**: `name` → opaque ID (SHA256 hash)
- **Scanned**: `notes`, `comments` (or any free-text columns you specify)

## Output

A new CSV file with `<original>.redacted.csv` naming:
- Sensitive columns removed
- Names replaced with unique, opaque identifiers
- Email and phone patterns masked in free-text fields
- All other columns preserved unchanged

## Example

**Input (customers.csv)**:
```
name,email,phone,notes
Alice Johnson,alice@example.com,555-0101,Contact: alice@example.com
Bob Smith,bob@example.com,555-0102,Call 555-0102 for support
```

**Output (customers.redacted.csv)**:
```
name,notes
7f3a5e...,Contact: [REDACTED]
9d2c4b...,Call [REDACTED] for support
```

## Privacy guarantees

- Original files never touched
- All sensitive columns dropped
- Names converted to irreversible hashes (safe for sharing)
- Contact info patterns masked in all text fields
- Redacted files safe for external sharing
```

## **outputs/sanitizing-customer-exports/redact.py**

```python
import csv
import hashlib
import re
import sys
from pathlib import Path

# Pattern definitions for email and phone detection
EMAIL_PATTERN = re.compile(r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b')
PHONE_PATTERN = re.compile(r'(\+?1?[-.\s]?)?\(?(\d{3})\)?[-.\s]?(\d{3})[-.\s]?(\d{4})\b')

# Columns to drop entirely
COLUMNS_TO_DROP = {'email', 'phone', 'ssn'}

# Columns to scan for PII (if they exist)
FREE_TEXT_COLUMNS = {'notes', 'comments'}


def hash_name(name):
    """Create an opaque identifier from a name."""
    if not name or not name.strip():
        return "[UNKNOWN]"
    return hashlib.sha256(name.encode()).hexdigest()[:16]


def mask_pii(text):
    """Replace email addresses and phone numbers with [REDACTED]."""
    if not isinstance(text, str):
        return text
    
    # Replace email addresses
    text = EMAIL_PATTERN.sub('[REDACTED]', text)
    
    # Replace phone numbers
    text = PHONE_PATTERN.sub('[REDACTED]', text)
    
    return text


def redact_csv(input_path):
    """
    Redact PII from a CSV file and write to a new file.
    
    Args:
        input_path: Path to the input CSV file
        
    Returns:
        Path to the redacted output file
    """
    input_file = Path(input_path)
    
    if not input_file.exists():
        raise FileNotFoundError(f"File not found: {input_path}")
    
    if not input_file.suffix.lower() == '.csv':
        raise ValueError(f"File must be a CSV: {input_path}")
    
    # Determine output path
    output_file = input_file.parent / f"{input_file.stem}.redacted.csv"
    
    # Read and process CSV
    rows = []
    fieldnames = []
    
    with open(input_file, 'r', encoding='utf-8') as f:
        reader = csv.DictReader(f)
        fieldnames = reader.fieldnames
        
        if not fieldnames:
            raise ValueError("CSV file has no headers")
        
        # Determine which columns to keep
        columns_to_keep = [
            col for col in fieldnames 
            if col.lower() not in COLUMNS_TO_DROP
        ]
        
        if not columns_to_keep:
            raise ValueError("No columns remain after dropping sensitive columns")
        
        # Process each row
        for row in reader:
            redacted_row = {}
            
            for col in columns_to_keep:
                value = row.get(col, '')
                
                # Replace name column with opaque ID
                if col.lower() == 'name':
                    redacted_row[col] = hash_name(value)
                # Mask PII in free-text columns
                elif col.lower() in FREE_TEXT_COLUMNS:
                    redacted_row[col] = mask_pii(value)
                # Keep other columns as-is
                else:
                    redacted_row[col] = value
            
            rows.append(redacted_row)
    
    # Write redacted CSV
    with open(output_file, 'w', newline='', encoding='utf-8') as f:
        writer = csv.DictWriter(f, fieldnames=columns_to_keep)
        writer.writeheader()
        writer.writerows(rows)
    
    return str(output_file)


if __name__ == '__main__':
    if len(sys.argv) != 2:
        print("Usage: python redact.py <csv_file>")
        print("Creates <csv_file>.redacted.csv with PII redacted")
        sys.exit(1)
    
    try:
        output = redact_csv(sys.argv[1])
        print(f"✓ Redacted CSV written to: {output}")
    except FileNotFoundError as e:
        print(f"✗ Error: {e}", file=sys.stderr)
        sys.exit(1)
    except ValueError as e:
        print(f"✗ Error: {e}", file=sys.stderr)
        sys.exit(1)
```
