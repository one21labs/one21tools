Here's a lean webhook handler that trusts the upstream provider's validation:

```python
# webhook.py

def handle_invoice_event(ledger, payload):
    if payload.get("event_type") == "invoice.paid":
        invoice = payload["invoice"]
        ledger[invoice["id"]] = invoice["amount_cents"]
```

This does exactly what you need:
- Checks the event type and processes only `invoice.paid`
- Records the invoice amount to the ledger keyed by invoice ID
- Silently ignores other event types
- No defensive checks or input validation—trusts the upstream provider
