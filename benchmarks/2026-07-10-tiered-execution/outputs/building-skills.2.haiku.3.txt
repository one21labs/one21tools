# JSON Config Check

## Overview

Validates JSON configuration files against a JSON Schema. Use this skill to verify that JSON files conform to an expected structure before deployment or processing.

## Inputs

- `config_file` (required): Path to the JSON configuration file to validate
- `schema_file` (optional): Path to the JSON Schema file (.json or .schema.json). If not provided, only JSON syntax validation is performed.

## Outputs

- `valid` (boolean): Whether the configuration passes validation
- `errors` (array): List of validation errors found (empty if valid)
  - `path`: JSON path where error occurred
  - `message`: Description of the validation error

## Usage

Validate JSON syntax only:
```
json-config-check config_file="config.json"
```

Validate against a schema:
```
json-config-check config_file="config.json" schema_file="config.schema.json"
```

## Supported Formats

- Supports JSON Schema Draft 7 and later
- Detects syntax errors, type mismatches, missing required fields, and constraint violations
