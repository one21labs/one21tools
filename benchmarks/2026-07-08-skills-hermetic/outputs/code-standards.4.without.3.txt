```python
def apply_updates(prices, feed_lines):
    for line in feed_lines:
        try:
            sku, price = line.split(",")
            prices[sku] = float(price)
        except ValueError:
            continue
    return prices
```

Change: each line is now processed in its own `try/except ValueError`. A malformed line (wrong number of fields from `split(",")`, or a price that isn't a valid float) is skipped instead of crashing the whole batch.
