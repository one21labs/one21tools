# CLAUDE.md

Guidance for Claude Code when working in this repository.

## Project Overview

`checkout-service` — Next.js 14 (App Router) + TypeScript + Tailwind, Prisma on Postgres.
Owned by the payments squad. Originally a 2021 Rails prototype, fully rewritten by the
platform team in 2023 — no Rails or legacy code remains; treat this as a native TS/Next app.

## Setup

Standard npm workflow. Full onboarding steps (clone, install, env, dev server) are in
`README.md` — follow that rather than duplicating steps here.

## Critical Rules

- **Money is always integer cents.** Never use floats for monetary values. A float-rounding
  bug in 2024 corrupted invoices — this is a hard rule, not a style preference.
- **Migrations always go through `./scripts/migrate.sh`.** Never run `prisma migrate`
  directly — it breaks the shadow database CI depends on.

## Testing

- `npm test` runs the Playwright E2E suite. It requires a running dev server and will fail
  without one — start `npm run dev` first.
- `npm run test:unit` runs unit tests and does not require a dev server.

## Auth

API routes authenticate via a custom `X-Session-Token` header scheme — **not** standard
Bearer tokens. Full scheme details: `docs/api.md`.

## Code Style

Full style guide: `docs/style-guide.md`. Notable deviations from typical TypeScript style:

- No default exports — use named exports only.
- Zod schemas are colocated next to their route handlers, not in a shared schemas directory.

## Deploy Process

Same procedure across all six company repos. Do not skip steps or reorder:

1. Merge to `main`
2. Wait for CI to go green
3. `npm run build:prod`
4. `./deploy staging`
5. Smoke-test `/healthz`
6. Page on-call if p95 latency exceeds 500ms
7. `./deploy prod`
8. Announce in `#releases`

Deploys are high-blast-radius — confirm with the user before running any deploy step rather
than executing the sequence autonomously.
