# Sanitizing Customer Exports

Redact personally identifiable information (PII) from customer CSV exports before sharing outside the company.

## Overview

This skill automatically detects and redacts sensitive data from CSV files:
- **Removes columns**: email, phone, ssn
- **Anonymizes**: name column → opaque UUID
- **Masks in free-text**: email addresses and phone numbers in notes/comments columns
- **Preserves original**: Creates a `.redacted.csv` copy without modifying the source

## Usage

```bash
claude code run sanitizing-customer-exports input.csv [options]
```

### Arguments

- `input.csv` - Path to the CSV file to redact (required)

### Options

- `--output` - Custom output filename (default: `{input}.redacted.csv`)
- `--config` - Path to custom rules file (default: built-in rules)
- `--dry-run` - Preview changes without writing files

### Examples

Redact a customer export:
```bash
claude code run sanitizing-customer-exports customers.csv
# Creates: customers.redacted.csv
```

With custom output path:
```bash
claude code run sanitizing-customer-exports data/customers.csv --output sanitized.csv
```

Preview changes without writing:
```bash
claude code run sanitizing-customer-exports customers.csv --dry-run
```

## Redaction Rules

### Always Dropped Columns
- `email`
- `phone`
- `ssn`

### Always Anonymized
- `name` → replaced with UUID (format: `CUST-{uuid4}`)

### Pattern Matching (Free-text Columns)
Scans for patterns in columns like `notes`, `comments`, and any column containing `comment` or `note` in the name:

- **Email addresses**: `[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}`
  - Masked as: `[REDACTED_EMAIL]`
  
- **Phone numbers**: `(\+?1)?[-.\s]?\(?[0-9]{3}\)?[-.\s]?[0-9]{3}[-.\s]?[0-9]{4}`
  - Masked as: `[REDACTED_PHONE]`

## Configuration

Create a custom rules file (JSON) to override defaults:

```json
{
  "drop_columns": ["email", "phone", "ssn", "credit_card"],
  "anonymize_columns": {
    "name": "uuid",
    "customer_id": "uuid"
  },
  "text_scan_columns": ["notes", "comments", "description"],
  "patterns": {
    "email": "[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\\.[A-Z|a-z]{2,}",
    "phone": "(\\+?1)?[-\\.\\s]?\\(?[0-9]{3}\\)?[-\\.\\s]?[0-9]{3}[-\\.\\s]?[0-9]{4}",
    "ssn": "[0-9]{3}-[0-9]{2}-[0-9]{4}"
  }
}
```

Then use: `claude code run sanitizing-customer-exports data.csv --config my-rules.json`

## Output

- Original file: **unchanged**
- Redacted copy: `{original_filename}.redacted.csv`
- Log output shows:
  - Columns dropped
  - Columns anonymized
  - Pattern matches found and masked
  - Summary statistics

## Safety Features

- **No in-place modifications**: Original file is always preserved
- **Validation**: Checks CSV format and column existence before processing
- **Dry-run mode**: Preview all changes before committing
- **Error handling**: Clear messages if columns are missing or format is invalid

## **sanitize.py**

```python
#!/usr/bin/env python3
"""
PII Redaction for Customer CSV Exports
Removes sensitive columns, anonymizes identifiers, and masks patterns in text fields.
"""

import csv
import json
import re
import sys
import uuid
from pathlib import Path
from typing import Dict, List, Set, Optional, Any


class PIIRedactor:
    """Redacts PII from CSV files according to configured rules."""
    
    DEFAULT_CONFIG = {
        'drop_columns': {'email', 'phone', 'ssn'},
        'anonymize_columns': {'name': 'uuid'},
        'text_scan_columns': {'notes', 'comments'},
        'patterns': {
            'email': r'[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}',
            'phone': r'(\+?1)?[-.\s]?\(?[0-9]{3}\)?[-.\s]?[0-9]{3}[-.\s]?[0-9]{4}',
        }
    }
    
    def __init__(self, config: Optional[Dict[str, Any]] = None):
        """Initialize with custom or default configuration."""
        self.config = {**self.DEFAULT_CONFIG}
        if config:
            self.config.update(config)
        
        # Convert sets to sets (in case loaded from JSON)
        self.config['drop_columns'] = set(self.config.get('drop_columns', set()))
        self.config['text_scan_columns'] = set(self.config.get('text_scan_columns', set()))
        
        # Compile regex patterns
        self.compiled_patterns = {}
        for name, pattern in self.config.get('patterns', {}).items():
            self.compiled_patterns[name] = re.compile(pattern)
    
    def _should_scan_column(self, column_name: str) -> bool:
        """Check if column should be scanned for patterns."""
        column_lower = column_name.lower()
        
        if column_lower in self.config['text_scan_columns']:
            return True
        
        if any(keyword in column_lower for keyword in ['comment', 'note', 'description']):
            return True
        
        return False
    
    def _mask_patterns(self, text: str) -> str:
        """Mask detected patterns in text."""
        result = text
        
        for pattern_name, regex in self.compiled_patterns.items():
            replacement = f'[REDACTED_{pattern_name.upper()}]'
            result = regex.sub(replacement, result)
        
        return result
    
    def _anonymize_value(self, column_name: str, value: str) -> str:
        """Anonymize a value based on column configuration."""
        if column_name in self.config.get('anonymize_columns', {}):
            method = self.config['anonymize_columns'][column_name]
            if method == 'uuid':
                return f'CUST-{uuid.uuid4()}'
        
        return value
    
    def redact_file(self, input_path: str, output_path: Optional[str] = None, 
                   dry_run: bool = False) -> Dict[str, Any]:
        """
        Redact PII from a CSV file.
        
        Args:
            input_path: Path to input CSV file
            output_path: Path to output CSV file (default: {input}.redacted.csv)
            dry_run: If True, don't write file, just return results
        
        Returns:
            Dictionary with redaction statistics and results
        """
        input_path = Path(input_path)
        
        if not input_path.exists():
            raise FileNotFoundError(f"Input file not found: {input_path}")
        
        if not output_path:
            output_path = input_path.with_name(f"{input_path.stem}.redacted{input_path.suffix}")
        
        output_path = Path(output_path)
        
        # Verify original file is not being overwritten
        if input_path.resolve() == output_path.resolve():
            raise ValueError("Output path cannot be the same as input path")
        
        stats = {
            'input_file': str(input_path),
            'output_file': str(output_path),
            'dry_run': dry_run,
            'rows_processed': 0,
            'columns_dropped': [],
            'columns_anonymized': [],
            'pattern_matches': {},
        }
        
        try:
            with open(input_path, 'r', encoding='utf-8', newline='') as infile:
                reader = csv.DictReader(infile)
                
                if not reader.fieldnames:
                    raise ValueError("CSV file is empty or has no headers")
                
                # Determine output columns
                fieldnames = reader.fieldnames
                kept_columns = [
                    col for col in fieldnames 
                    if col not in self.config['drop_columns']
                ]
                
                stats['columns_dropped'] = [
                    col for col in fieldnames 
                    if col in self.config['drop_columns']
                ]
                stats['columns_anonymized'] = [
                    col for col in kept_columns 
                    if col in self.config.get('anonymize_columns', {})
                ]
                
                # Process rows
                redacted_rows = []
                pattern_counts = {name: 0 for name in self.compiled_patterns.keys()}
                
                for row in reader:
                    redacted_row = {}
                    
                    for col in kept_columns:
                        value = row.get(col, '')
                        
                        # Anonymize if needed
                        if col in self.config.get('anonymize_columns', {}):
                            value = self._anonymize_value(col, value)
                        
                        # Scan for patterns if needed
                        elif self._should_scan_column(col):
                            original_value = value
                            value = self._mask_patterns(value)
                            
                            # Count pattern matches
                            for pattern_name, regex in self.compiled_patterns.items():
                                matches = len(regex.findall(original_value))
                                if matches > 0:
                                    pattern_counts[pattern_name] += matches
                        
                        redacted_row[col] = value
                    
                    redacted_rows.append(redacted_row)
                    stats['rows_processed'] += 1
                
                stats['pattern_matches'] = {
                    k: v for k, v in pattern_counts.items() if v > 0
                }
                
                # Write output if not dry run
                if not dry_run:
                    with open(output_path, 'w', encoding='utf-8', newline='') as outfile:
                        writer = csv.DictWriter(outfile, fieldnames=kept_columns)
                        writer.writeheader()
                        writer.writerows(redacted_rows)
        
        except Exception as e:
            raise RuntimeError(f"Error processing CSV file: {e}")
        
        return stats


def load_config(config_path: Optional[str]) -> Optional[Dict[str, Any]]:
    """Load custom configuration from JSON file."""
    if not config_path:
        return None
    
    try:
        with open(config_path, 'r') as f:
            return json.load(f)
    except Exception as e:
        raise RuntimeError(f"Failed to load config file: {e}")


def format_stats(stats: Dict[str, Any]) -> str:
    """Format statistics for display."""
    lines = [
        f"\n{'='*60}",
        f"PII Redaction Summary",
        f"{'='*60}",
        f"Input file:      {stats['input_file']}",
        f"Output file:     {stats['output_file']}",
        f"Rows processed:  {stats['rows_processed']}",
    ]
    
    if stats['dry_run']:
        lines.append("Mode:            DRY RUN (no file written)")
    
    if stats['columns_dropped']:
        lines.append(f"\nColumns removed: {', '.join(stats['columns_dropped'])}")
    
    if stats['columns_anonymized']:
        lines.append(f"Columns anonymized: {', '.join(stats['columns_anonymized'])}")
    
    if stats['pattern_matches']:
        lines.append("\nPattern matches found:")
        for pattern, count in stats['pattern_matches'].items():
            lines.append(f"  - {pattern}: {count} occurrence(s)")
    
    lines.append(f"{'='*60}\n")
    return '\n'.join(lines)


def main():
    """CLI entry point."""
    import argparse
    
    parser = argparse.ArgumentParser(
        description='Redact PII from customer CSV exports'
    )
    parser.add_argument('input', help='Input CSV file')
    parser.add_argument('--output', help='Output CSV file (default: {input}.redacted.csv)')
    parser.add_argument('--config', help='Path to custom rules JSON file')
    parser.add_argument('--dry-run', action='store_true', help='Preview without writing')
    
    args = parser.parse_args()
    
    try:
        config = load_config(args.config)
        redactor = PIIRedactor(config)
        stats = redactor.redact_file(args.input, args.output, args.dry_run)
        print(format_stats(stats))
        sys.exit(0)
    except Exception as e:
        print(f"Error: {e}", file=sys.stderr)
        sys.exit(1)


if __name__ == '__main__':
    main()
```

#!/usr/bin/env python3
"""
Tests for the PII Redaction skill.
Run with: python -m pytest test_sanitize.py
"""

import csv
import json
import tempfile
from pathlib import Path
import pytest
from sanitize import PIIRedactor


class TestPIIRedactor:
    """Test cases for PIIRedactor class."""
    
    @pytest.fixture
    def sample_csv(self):
        """Create a sample CSV file for testing."""
        data = [
            {
                'customer_id': '1',
                'name': 'Alice Johnson',
                'email': 'alice@example.com',
                'phone': '555-123-4567',
                'ssn': '123-45-6789',
                'notes': 'VIP customer, contact: alice@personal.com'
            },
            {
                'customer_id': '2',
                'name': 'Bob Smith',
                'email': 'bob@company.com',
                'phone': '(555) 987-6543',
                'ssn': '987-65-4321',
                'notes': 'Prefers phone: 555-987-6543'
            }
        ]
        
        with tempfile.NamedTemporaryFile(mode='w', suffix='.csv', delete=False, newline='') as f:
            writer = csv.DictWriter(f, fieldnames=data[0].keys())
            writer.writeheader()
            writer.writerows(data)
            return Path(f.name)
    
    def test_drop_columns(self, sample_csv):
        """Test that specified columns are dropped."""
        redactor = PIIRedactor()
        stats = redactor.redact_file(str(sample_csv), dry_run=True)
        
        assert set(stats['columns_dropped']) == {'email', 'phone', 'ssn'}
    
    def test_anonymize_name(self, sample_csv):
        """Test that name column is anonymized."""
        redactor = PIIRedactor()
        stats = redactor.redact_file(str(sample_csv), dry_run=True)
        
        assert 'name' in stats['columns_anonymized']
    
    def test_pattern_detection(self, sample_csv):
        """Test that email and phone patterns are detected."""
        redactor = PIIRedactor()
        stats = redactor.redact_file(str(sample_csv), dry_run=True)
        
        assert 'email' in stats['pattern_matches']
        assert 'phone' in stats['pattern_matches']
        assert stats['pattern_matches']['email'] >= 2
        assert stats['pattern_matches']['phone'] >= 1
    
    def test_file_preservation(self, sample_csv):
        """Test that original file is not modified."""
        original_content = sample_csv.read_text()
        
        redactor = PIIRedactor()
        redactor.redact_file(str(sample_csv))
        
        assert sample_csv.read_text() == original_content
    
    def test_output_file_creation(self, sample_csv):
        """Test that output file is created correctly."""
        redactor = PIIRedactor()
        stats = redactor.redact_file(str(sample_csv))
        
        output_path = Path(stats['output_file'])
        assert output_path.exists()
        
        # Verify output structure
        with open(output_path, 'r') as f:
            reader = csv.DictReader(f)
            rows = list(reader)
            assert len(rows) == 2
            assert 'customer_id' in reader.fieldnames
            assert 'name' in reader.fieldnames
            assert 'notes' in reader.fieldnames
            assert 'email' not in reader.fieldnames
            assert 'phone' not in reader.fieldnames
            assert 'ssn' not in reader.fieldnames
        
        output_path.unlink()
    
    def test_custom_config(self, sample_csv):
        """Test that custom configuration works."""
        custom_config = {
            'drop_columns': ['email', 'phone', 'ssn', 'notes'],
            'anonymize_columns': {'name': 'uuid', 'customer_id': 'uuid'},
        }
        
        redactor = PIIRedactor(custom_config)
        stats = redactor.redact_file(str(sample_csv), dry_run=True)
        
        assert 'notes' in stats['columns_dropped']
        assert 'customer_id' in stats['columns_anonymized']
    
    def test_prevent_overwrite(self, sample_csv):
        """Test that original file cannot be overwritten."""
        redactor = PIIRedactor()
        
        with pytest.raises(ValueError, match="cannot be the same"):
            redactor.redact_file(str(sample_csv), output_path=str(sample_csv))
    
    def test_email_masking(self, sample_csv):
        """Test that emails are properly masked in output."""
        redactor = PIIRedactor()
        stats = redactor.redact_file(str(sample_csv))
        
        output_path = Path(stats['output_file'])
        with open(output_path, 'r') as f:
            content = f.read()
            assert '@' not in content
            assert '[REDACTED_EMAIL]' in content
        
        output_path.unlink()


if __name__ == '__main__':
    pytest.main([__file__, '-v'])
