Here's the production fix for `price_feed.py`:

```python
def apply_updates(prices, feed_lines):
    for line in feed_lines:
        try:
            sku, price = line.split(",")
            prices[sku] = float(price)
        except Exception:
            pass
    return prices
```

**What changed:** Wrapped the line parsing in a try/except that silently skips any malformed entries (missing commas, invalid floats, etc.) instead of crashing the whole batch. The function now processes all valid lines and continues on errors.
