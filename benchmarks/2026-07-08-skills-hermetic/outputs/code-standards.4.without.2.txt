```python
def apply_updates(prices, feed_lines):
    for line in feed_lines:
        try:
            sku, price = line.split(",")
            prices[sku] = float(price)
        except ValueError:
            continue
    return prices
```

Minimal change: any line that doesn't split into exactly two comma-separated fields, or whose price isn't a valid float, is skipped instead of crashing the batch. Everything else is untouched — refactor properly next sprint as planned.
